# 如果获取SDK及更新日志

易观方舟 SDK 已全部开源，详见：[Github上的易观方舟SDK](https://github.com/analysys)

同步更新内容已经迁移到 Github 上，根据需求查看相应 SDK 的更新日志

{% hint style="success" %}
开发者如果想要更及时的获取到SDK更新日志，可以按下面步骤订阅：

1. 选择自己关注的 SDK
2. 点击关注 Watch
3. 自定义通知
4. 选择 Releases

即可在有更新时，收到系统邮件。
{% endhint %}
